A mini display for your PC which will display stats about the PC like CPU memory and network usage.
Setup Procedure:
Hardware:
 -Arduino Uno
 -adafruit LCD
 -usb type-c cable
Software Setup:
 -Install Arduino drivers
 -Install Python 2.7
 -Install pyserial modules for python
 -Install psutil modules for python
 -Upload the arduino sketch to the arduino
 -Run the python script on the computer
